# Slink-Snake-Game

A snake-style arcade game inspired by Slither.io, built with **Cocos2d-x** (C++).

---

## Gameplay

- Control your snake with the **mouse** — the snake always moves toward the cursor
- **Eat food** (glowing orbs) to grow longer
- **Hold left mouse button** to boost speed (requires length ≥ 10, snake shrinks while boosting)
- **Avoid the red border** — hitting it ends the game
- Your length is shown in the bottom-left corner and in the Leaderboard panel
